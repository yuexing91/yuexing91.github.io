# 介绍

Vuc3是一个基于Vue3.0的SFC模板语法的可视化代码编辑器（UI框架依赖Ant Design of Vue），可以快速的实现Vue组件的可视化编辑。

# 特性

# 基于AST操作

Vuc3是将Vue文件转成AST（抽象语法树）来操作的。

你可以像下面这样直接传入一个vue文件到vuc设计器中

let vueContent = `
<template>
  <div>{{ message }}</div>
</template>
<script>
export default {
  data() {
    return {
      message: 'Hello World!!'
    }
  }
};
</script>
`;

let app = createApp(SampleDesigner, { vueContent });
app.mount('#app');

# 快速接入已有组件

Vuc3允许任意组件接入，你只需提供一个简单的配置即可。

假设我们现在已经有了一个DemoButton组件，包含了按钮文本 text 的 props。


<template>
  <button>{{text}}</button>
</template>
<script>
export default {
  props: ['text'],
}
</script>

给DemoButton组件的加入配置

Configurator.setVucConfig({
  id: 'demo-button',
  props: {
    text: {
      label: '按钮标签',
      editors: 'string',
    },
  },
});

完整代码如下：

let vueContent = `
<template>
  <DemoButton :text="message"></DemoButton>
</template>
<script>
const DemoButton = {
  template:'<button>{{text}}</button>',
  props:['text'],
}  
export default {
  components:{
    DemoButton
  },
  data() {
    return {
      message: 'Hello World!!'
    }
  }
};
</script>
`;

Configurator.setVucConfig({
  id: 'demo-button',
  props: {
    text: {
      label: '按钮标签',
      editors: 'string',
    },
  },
});

# 组合式组件

实际使用中经常会碰到需要将多层级组件做为一个整体来操作，此时就可以使用组合式组件来配置。例如在使用栅格布局的表单中单个表单项是由 col>formitem>input 这个结构组成的，我们希望能够将其作为一个整体而无需进行单独封装。


<a-col :col="{span:6}">
  <a-from-item name="name" label="名称">
    <a-input v-model:value="fromData.name"></a-input>
  </a-from-item>
</a-col>

上述例子，我们有下面几个要求：

  1. a-col、a-from-item、a-input等组件可以同时进行配置。
  2. a-from-item 的 name 属性改变是 a-input 的 v-mode:value 同时也改变。
