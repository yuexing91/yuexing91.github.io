# 组件配置

用来描述 vuc 中需要配置的组件信息，通过 Configurator.setVucConfig 使用，传入一个普通的 js 对象。

# id

组件的标签名称。

  • 类型： string | array

  • 必须

# name

显示名称。

  • 类型： string

# type

描述组件的类型。

{
  "type": ["input"]
}
  • 类型： array

# eventTypes

组件事件配置。

  • 类型： array
    • id ： string 事件名称
    • name ： string 显示名称
export default {
  eventTypes: [
    {
      id: 'click',
      name: '点击',
    },
  ]
}

# slots

插槽配置。

  • 类型： array
    • slot ： string 插槽名称
    • name ： string 显示名称
export default {
  slots: [
    {
      slot: 'default',
      name: '按钮内容',
    },
  ]
}

# props

props配置，object 格式，key 为 prop 的 name，value为具体的配置项。

  • 类型： object

    • key ： prop 的 name。
    • value ： object prop 的具体配置。
      • label ： string prop 的显示名称。
      • editors ： string | object | array 内容编辑器配置。

  • 示例：

export default {
  props: {
    type: {
      label: '按钮类型',
      editors: 'select?@items=default|primary|dashed|text|danger|link'
    },
    size: {
      label: '按钮尺寸',
      editors: [
        {
          name: '选项',
          type: 'radio-group',
          props: {
            items: [
              {
                value: 'large',
                label: '大',
              },
              {
                value: 'default',
                label: '中',
              },
              {
                value: 'small',
                label: '小',
              },
            ],
          },
        }
      ]
    }
  }
}

# vModel

  • 类型： string | array
  • 详细：

配置 v-model 选项 { vModel:'value' } 相当于 v-model:value

# text

文本编辑选项，例如按钮组件 <Button>按钮</Button> 一般都需要能够设置按钮文本内容。

  • 类型： object
    • label string 文本编辑标签

# parentLimit

拖拽时父级容器限制，例如 Col 组件一般只能放置在 Row 组件内。

  • 类型： function

# childLimit

拖拽时子级组件限制，例如 Row 组件一般只能包含 Col 组件内。

  • 类型： function

# vucEditorOption

vucEditor 配置。

  • 类型： object
    • onBeforeSelectNode : boolean | function 鼠标选中之前钩子，返回 false 取消选中。

# vucProxyOption

组件代理的配置。

  • 类型： object
    • textProxy : boolean 是否需要代理文本
    • onBeforeRender : function 渲染之前的钩子
    • onRender : function 渲染时的钩子
