# 操作手册

# 组件资源管理视图

  • 按下鼠标拖拽组件，拖拽的行为受 被拖拽组件的 parentLimit （可被放置的目标） 及目标组件的 slots （是否拥有插槽）、childLimit（允许放置的组件） 等配置影响。
  • 双击，将所选组件插入到当前组件的内部（所选组件与当前组件不同时）或后面（所选组件与当前组件相同时）

# 编辑器

# 操作

  • 在上按下鼠标开始拖拽。
  • 在组件上右键弹出操作菜单。
    • 移动，开始拖拽无需一直按住鼠标左键。
    • 前移
    • 后移
    • 删除
    • 复制
    • 剪切
    • 重复，在当前组件后面复制一个当前组件。
    • 粘贴，将复制\剪切的组件粘贴到当前组件的前方\后方。
    • 粘贴到插槽，将复制\剪切的组件粘贴到当前组件的插槽。

# 快捷键

  • Ctrl + C 复制
  • Ctrl + X 剪切
  • Ctrl + V 粘贴
  • Ctrl + I 插入到内部
  • Ctrl + P 重复
  • Alt + Up 向前移动
  • Alt + Left 向前移动
  • Alt + Down 向后移动
  • Alt + Right 向后移动
  • Delete 删除
  • Insert 插入到内部
  • Up 选中父节点
  • Down 选中子节点
  • Left 选中前一个节点
  • Right 选中后一个节点
  • Esc 取消选中

# 组件配置

# props

组件的 props 可以存在多种配置类型，例如 width 的值可以是 数字 :width="100"、字符串 width="100px"、变量 :width="width" 或者也可以是表达式 :width="100 + 'px'"。
