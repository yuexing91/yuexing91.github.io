# 内容编辑器

内容编辑器是 vuc 用来编辑代码的核心功能，自带了 string 字符串、boolean 布尔值、number 数字、 select 下拉框、radio-group 单选框、array 数组、expression 表达式。

你还可以通过 Configurator.registerValueEditor 来注册自己的编辑器。

# 在 vueConfig 的 props 配置中使用

export default {
  props: {
    type: {
      label: '按钮文本',
      editors: 'string'
    }
  },
  ...,
}

  • editors 的值为字符串时，可使用 url 参数语法来配置。

    • 格式：{editorId}?{arg1}={arg1Value}&{arg2}={arg2Value}，
    • 例如：select?@items=default|primary|dashed|text|danger|link

  • editors 的值为对象时，需要包含 name（显示名称），type（编辑器ID），props（需要传给编辑器组件的props）

( {
  name: '选项',
  type: 'radio-group',
  props: {
    items: [
      {
        value: 'large',
        label: '大',
      },
      {
        value: 'default',
        label: '中',
      },
      {
        value: 'small',
        label: '小',
      },
    ],
  },
} )
  • editors 的值为数组时，可以任意将以上2种格式组合。

# 内置编辑器

# 文本 string

用于编辑文本内容，可以通过传入 items 作为可加入自动完成项。

  • props
    • items Array<{value:string,label:string}> | function(text) 选项组。

# 布尔 boolean

用于编辑布尔值。

  • props
    • trueValue string 选中时的值
    • falseValue string 未选中时的值
    • trueText string 选中时的文本
    • falseText string 未选中时的文本
    • defaultValue string 默认值

# 数字 number

  • props
    • min number 最小值范围
    • max number 最大值范围

# 下拉框 select

  • props
    • items Array<{value:string,label:string}> 选项组。

# 单选框组 radio-group

  • props
    • items Array<{value:string,label:string}> 选项组。

# 数组 array

  • props
    • type string 类型，object表示编辑对象数据，否则为字符串。
    • isTreeData boolean 是否树形数据。
    • fields Array<{label:string,field:string,editor:string}> 数组中字段列表。
    • keyName string 对应的 fields 中唯一 field 字段。

# 表达式 expression

  • props
    • inline boolean 是否为行内模式。

# 自定义内容编辑器

有时内置编辑器可能不能满足程序的需求，例如在 ant-design-vue 的 a-from 表单组件中，我们需要给的表单的标签设置一个固定宽度， 需要将label-col (opens new window)的属性值 设置为:label-col="{ style: { width: '150px' } }"，我们可以通过自定义内容编辑器来实现。如下图：

  1. 提供一个带有 v-model:value 编辑器组件 ColWidth.vue。

<template>
  <a-input v-model:value="val"/>
</template>

<script>
export default {
  emits: 'update:value',
  props: {
    value: String,
  },
  computed: {
    val: {
      get() {
        if (this.value) {
          let match = this.value.match(../images/width\s*:\s*(["'])(.*?)\1/);
          if (match) {
            return match[2];
          }
        }
        return '';
      },
      set(v) {
        this.$emit('update:value', `{ style: { width: '${ v }' } }`)
      }
    }
  },
};
</script>
  1. 通过Configurator.registerValueEditor注册该组件。
import ColWidth from './ColWidth.vue';

Configurator.registerValueEditor('colWidth', {
  name: '栅格宽度',
  dataType: 'object',
  Editor: ColWidth,
});
  1. 给 a-form 组件的 label-col 配置编辑器。
export default {
  props: {
    labelCol: {
      name: '标签宽度',
      editors: 'colWidth'
    },
    ...
  }
}
