# 组合式组件

将多个组件做为一个整体进行操作。

# 注册组合组件

可通过 Configurator.registerComboType。

Configurator.registerComboType({
  id: 'FormItem',
  name: '表单项',
  template: `<a-form-item key="formItem">
      <type.input key="input" allowChildren="true"/>
    </a-form-item>`,
  configs() {
    return [
      {
        title: '属性',
        mode: 'prop',
      },
      {
        title: '事件',
        mode: 'event',
      },
      {
        title: '样式',
        mode: 'style',
      },
      //... 更多
    ];
  },
});

配置解释

  • id string组合的唯一ID
  • name string 组合名称
  • template string 组合的模板结构，只允许存在一个根节点。
    • 节点标签可以是组件的id值或者可以是以 type.开头后面接组件的类型。
    • 属性 key 设置节点的 key 值。
    • 属性allowChildren 设置是否可以包含子节点。
    • 上述实例中表示的是匹配包含input类型的且允许存在子节点的a-form-item节点，如<a-form-item> <a-input/> </a-form-item>。
  • configs function(combo) 组合的配置信息。
    • return Array<{title:string,mode:comboMode,...others}>
      • title 配置模块名称。
      • comboMode 组合的配置模块。
      • others 不同模块需要的不同的参数配置。

# 内置组合配置模块

# prop

配置组合式组件中各节点的 props，对应的 vuc-config 中的 props 属性。

[
  {
    title: '属性',
    mode: 'prop',
    items: [
      {
        nodeKey: 'formItem',
        name: 'name',
        label: '字段名',
      },
    ],
  },
  //... 其他模块
]
  • items Array<nodeKey:string,name:string,label:string,excludeTags:Array<string>>
    • nodeKey 需要配置的节点 key，对应 template 中的 key。
    • name 组件 prop 的值，对应 vucConfig.props 中的属性名。如果 vucConfig.props 中找不到对应的配置则不显示。
    • label 显示名称，不配置则从vucConfig中取。
    • excludeTags 排除的组件tag，如果 nodeKey 对应的组件标签包含在excludeTags中，则此项配置不显示。

# event

用于快速配置事件。

[
  {
    title: '事件',
    mode: 'event',
    items: [
      {
        nodeKey: 'input',
        id: 'change',
        name: '值变更事件',
      },
    ],
  },
  //... 其他模块
]
  • items Array<nodeKey:string,id:string,name:string,excludeTags:Array<string>>
    • nodeKey 需要配置的节点 key，对应 template 中的 key。
    • id 组件事件名称，对应 vucConfig.eventTypes 中的 id。如果 vucConfig.eventTypes 中找不到对应的配置则不显示。
    • name 显示名称，不配置则从vucConfig中取。
    • excludeTags 排除的组件tag，如果 nodeKey 对应的组件标签包含在excludeTags中，则此项配置不显示。

# style

用于快速配置常用样式。

[
  {
    title: '样式',
    mode: 'style',
    items: [
      {
        nodeKey: 'input',
        id: 'width',
        name: '宽度',
        excludeTags: ['a-checkbox-group', 'a-radio-group'],
      },
    ],
  },
  //... 其他模块
]
  • items Array<nodeKey:string,id:string,name:string,excludeTags:Array<string>>
    • nodeKey 需要配置的节点 key，对应 template 中的 key。
    • id 样式 id。
    • name 显示名称。
    • excludeTags 排除的组件tag，如果 nodeKey 对应的组件标签包含在excludeTags中，则此项配置不显示。

# 自定义组合配置模块

你可以通过 Configurator.registerComboMode 来自定义组合的配置模块。

# 参数

  • mode string 模块名称
  • Component Vue组件
  • parseComponentProps function(config, combo) 转换 Component 需要的 props

示例：

registerComboMode({
  mode: 'event',
  Component: QuickEventPanel,
  parseComponentProps(config, combo) {
    return parse(config.items);
  },
})
