# 内置组件

# SampleDesigner

一个简单的设计器组件，用以快速创建一个 vuc 设计器。

# props

  • vueContent
    • type string vue文件内容
  • editorOptions
    • type object 编辑器的配置信息
  • explorerComponents
    • type array 组件资源管理器信息

# 插槽

  • toolbar 工具栏 。
    • 参数 { vucAst, selectNode }
  • left 设计器左侧面板。
    • 参数 { vucAst, selectNode }
  • right 设计器右侧面板。
    • 参数 { vucAst, selectNode }

# SampleNodePanel

对应 SampleDesigner 主要用于配置 vucNode 组件选项的面板。 包括 combo-panel、text-panel、props-panel、quick-event-panel 、event-panel、style-panel、directive-panel等内容。

# SampleOptionPanel

对应 SampleDesigner 主要用于配置页面选项的面板。包括 data-panel、computed-panel、method-panel、watch-panel。

# Designer

  • editorOptions

# 插槽

  • right 设计器右侧面板。
    • 参数 { vucAst, selectNode }

# ExplorerView

# props

  • components
    • type array 组件列表

# PanelView

常规视图。

# props

  • title
    • type string 面板标题

# PropsPanel

VucNode的 props 配置。

# props

  • vucNode
    • type VucNode 需要配置的VucNode

# PropItem

VucNode的单项 props 配置。

# props

  • name
    • type string 需要配置的prop的name
  • label
    • type string 需要配置的prop的显示标签
  • config
    • type object 需要配置的prop的配置属性，为空则从 vucConfig 的配置中读取
  • vucNode
    • type VucNode 需要配置的VucNode
