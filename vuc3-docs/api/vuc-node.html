# VucNode

通过 createVucNode(template) 创建用于描述 vue template 中的节点信息的对象。

# 实例方法

# setAttrValue

  • 参数：
    • name 属性名称
    • value 属性值
  • 返回值：
    • 发生了改变返回 true 否则为false
  • 用法：
    • 设置节点属性。
  • 示例：
node = createVucNode(`<a-input/>`)
node.setAttrValue('v-model:value', 'name') // true
node.setAttrValue('v-model:value', 'name') // false

# delAttr

  • 参数：
    • nameOrIndex
  • 用法：
    • 删除节点属性。
  • 示例：
node.delAttr('v-model:value')
node.delAttr(0)

# getAttr

  • 参数
    • name
  • 用法：
    • 获取节点的 VucNodeAttr 对象。

# getAttrValue

  • 参数
    • name
  • 用法：
    • 获取节点 name 的值。

# getDirectives

  • 用法：
    • 返回节点绑定的所有指令。

# getDirective

  • 参数

    • name
    • arg 可选

  • 用法：

    • 返回节点指定 name 指令。

  • 示例：

node.getDirective('v-model')
node.getDirective('v-model', 'value')

# getBind

  • 参数

    • name

  • 用法：

    • 返回节点绑定属性（包括静态、动态绑定）。
node = createVucNode(`<a-button size="small" :type="type" v-bind:shape="shape"/>`)
node.getBind("size") // size="small"
node.getBind("type") // :type="type"
node.getBind("shape") // :shape="shape"

# getStaticBind

  • 参数

    • name

  • 用法：

    • 返回节点静态绑定属性。
node = createVucNode(`<a-button size="small"/>`)
node.getStaticBind("size") //:type="type"

# getVBind

  • 参数
    • name
  • 用法：
    • 返回节点动态绑定属性。

# getOn

  • 参数
    • arg
  • 用法：
    • 返回 on 指令属性。

# getSlot

  • 用法：
    • 返回 slot 指令属性。

# delSlot

  • 用法：
    • 删除 slot 指令属性。

# getSlotName

  • 用法：
    • 获取 slot 名称。

# getSlotParam

  • 用法：
    • 获取 slot 参数。

# getBelongSlotName

  • 用法：
    • 获取当前节点位于父组件的插槽。

# isText

  • 用法：
    • 是否为文本节点。

# getAttrMap

  • 用法：
    • 以 map 的方式返回节点所有属性。

# getAttrsStr

  • 用法：
    • 以字符串的方式返回节点所有属性。

# getIndex

  • 用法：
    • 获取当前节点位于父节点的索引位置。

# getChildIndex

  • 参数
    • child
  • 用法：
    • 获取 child 节点的索引位置。

# getChild

  • 参数
    • index
  • 用法：
    • 获取指定 index 的节点。

# getBeforeNode

  • 用法：
    • 获取当前节点的前一个节点。

# getAfterNode

  • 用法：
    • 获取当前节点的后一个节点。

# getParentNode

  • 用法：
    • 获取当前节点的父节点。

# remove

  • 用法：
    • 删除当前节点。

# removeChild

  • 参数
    • child
  • 用法：
    • 获取指定 child 节点。

# insertNode

  • 参数
    • node
    • index
  • 用法：
    • 在指定 index 后插入单个node 节点。

# insertNodes

  • 参数
    • nodes
    • index
  • 用法：
    • 在指定 index 后插入多个nodes 节点。

# appendNode

  • 参数
    • nodes
  • 用法：
    • 在末尾后插入多个nodes 节点。

# insertBefore

  • 参数
    • beforeNode
  • 用法：
    • 在当前节点之前插入多个 beforeNode 节点。

# insertAfter

  • 参数
    • afterNode
  • 用法：
    • 在当前节点之后插入多个 afterNode 节点。

# insertToSlot

  • 参数
    • insertNode
    • slot
  • 用法：
    • 将 insertNode 插入到指定的 slot 插槽中。

# moveToBefore

  • 用法：
    • 将当前节点往前移动。

# moveToAfter

  • 用法：
    • 将当前节点往后移动。

# moveToSlot

  • 参数
    • slot
  • 用法：
    • 将当前节点移动到指定 slot 插槽中。

# eqTag

  • 参数
    • tag
  • 用法：
    • 当前节点标签是否等于指定的 tag 标签。

# setStyle

  • 参数
    • style
    • dynamic
  • 用法：
    • 设置当前节点的样式。

# setText

  • 参数
    • content
    • type
  • 用法：
    • 设置当前节点的文本节点。

# getText

  • 用法：
    • 获取当前节点的文本内容。

# clone

  • 用法：
    • 克隆当前节点。

# closest

  • 参数
    • selector
  • 用法：
    • 向上查找指定选择器。

# getParents

  • 用法：
    • 获取所有的父级元素列表。

# eachAllNode

  • 参数
    • iterator function
  • 用法：
    • 遍历所有子节点。

# dfs

  • 参数
    • iterator function
    • self boolean 是否需要遍历当前节点。
  • 用法：
    • 深度优先遍历，。

# bfs

  • 参数
    • iterator function
    • self boolean
  • 用法：
    • 广度优先遍历。

# querySelector

  • 参数
    • selector
  • 用法：
    • 根据选择器 selector 查找单个节点。

# querySelectorAll

  • 参数
    • selector
  • 用法：
    • 根据选择器 selector 查找所有满足条件的节点。

# findChild

  • 参数
    • selector
  • 用法：
    • 根据选择器 selector 查找单个节点（只在子节点中查找）。

# findChildren

  • 参数
    • selector
  • 用法：
    • 根据选择器 selector 查找所有满足条件的节点（只在子节点中查找）。

# getConfig

  • 参数
    • key
    • defaultValue

# isHTML

  • 用法：
    • 是否是HTML节点。

# isScopeSlot

  • 用法：
    • 是否为作用于插槽。

# isTemplateNode

  • 用法：
    • 是否为 template 节点。

# isOffspring

  • 参数
    • node
  • 用法：
    • 判断 node 是否是当前节点的子孙元素

# toTemplate

  • 用法：
    • 输出为 template 格式。
